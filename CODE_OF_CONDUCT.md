# Code of Conduct

## Our Pledge

We want the Constella server repository to be respectful, inclusive, and productive for everyone contributing to backend, API, collaboration, and infrastructure work.

## Expected Behavior

- Communicate clearly and respectfully.
- Focus feedback on technical issues, not people.
- Share evidence when reporting bugs or regressions.
- Be collaborative during review, especially on API and infrastructure changes.

## Unacceptable Behavior

- Harassment, discrimination, or abusive language.
- Personal attacks in issues, reviews, or discussions.
- Posting private credentials, tokens, or sensitive production details.
- Deliberately disruptive or bad-faith participation.

## Scope

This policy applies in:

- GitHub issues, pull requests, and discussions
- Review comments and documentation changes
- Community spaces directly related to this repository

## Enforcement

Maintainers may edit, hide, lock, or remove content that violates this policy, and may restrict participation when necessary.

For serious concerns, contact the repository maintainer privately through GitHub and include relevant links or evidence.
